    XML Based Identifiers Across Namespaces:

    More Detail:

    Some sources of file format identification offer alternatives to file format signatures. Tools like Siegfried can also provide alternative techniques too. These techniques output identifiers within the framework set out by DROID and Siegfried, as oppesed the Linux tool File. This listing presents a count of identification results from scanning files using Freedesktop's MIME XML matching mechanism. Freedesktop's XML matching mechanism will match features of an XML document once the identifying mechanism has established the file it is looking at is an XML file.
